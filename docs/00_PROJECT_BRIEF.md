# Project Brief

# Project Goal

Build a Python Bitcoin quant trading application that can eventually support backtesting, paper trading, and live trading.

# First-Version Scope

- market data contract
- CSV/local candle loading
- RSI strategy
- basic backtest
- paper trader

# First-Version Non-Goals

- live trading
- real order execution
- risk management
- dashboard
- database
- scheduler
- ML
- futures
- leverage

# High-Level Evolution Path

1. Codex working documents
2. Python project setup
3. market data contract
4. CSV/local data provider
5. RSI strategy
6. basic backtest
7. paper trader
8. Binance historical candle downloader
9. improved backtesting
10. paper trading with state
11. later risk management
12. later live trading
